SyntaxError: Non-ASCII character '\xe9'

python 报SyntaxError: Non-ASCII character ‘\xe9’

在Python中运行时出现
SyntaxError: Non-ASCII character ‘\xe9’ in file 异常……

Python的默认编码文件是用的ASCII码，你将文件存成了UTF-8也没用。
解决办法很简单：
只要在文件开头加入 # – coding: UTF-8 – 或者 #coding=utf-8 就行了。

2.采用手动创建

2.1在 hexo 主目录下 source -> _posts 新建以 .md 为后缀的文件。


Front-matter 是文件最上方以 — 分隔的区域，用于指定个别文件的变量，就是头部位置，举例来说：
- - -
title: Hello World
date: 2017/7/13 20:46:25
- - -

以下是预先定义的参数，您可在模板中使用这些参数值并加以利用。

2.2分类和标签

但是Hexo不支持指定多个同级分类。下面的指定方法：
categories:
- Diary
- Life
会使分类Life成为Diary的子分类，而不是并列分类。因此，有必要为您的文章选择尽可能准确的分类
